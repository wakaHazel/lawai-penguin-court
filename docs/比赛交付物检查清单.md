## 7. 提交材料命名约束

后续正式提交材料，文件名统一优先使用 `中文命名`，不要默认使用英文名。

建议命名风格：

- `项目简介.docx`
- `项目解决方案.docx`
- `AI工具使用说明（工作流与Prompt）.docx`
- `项目说明书.docx`
- `输入输出示例.docx`
- `作品演示视频.mp4`
- `答辩PPT.pptx`

补充要求：

- [ ] 提交包文件名统一使用中文
- [ ] 文件名简洁、直观、便于评委识别
- [ ] 同一批材料命名风格保持一致
- [ ] 避免中英混搭和随意缩写

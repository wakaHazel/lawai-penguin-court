# 区域初赛项目简介（企鹅法庭）V1

## 1. 项目基础信息

- 项目名称：`企鹅法庭·沉浸式庭审模拟与立法沙盘推演AI智能体系统`
- 赛题名称：`【D06】法律AI应用创新与实践`
- 项目类型：`系统解决方案`
- 应用方向：`智慧法律应用`
- 项目定位：
  - 面向法律学习、庭前准备、法律服务辅助等场景
  - 基于腾讯系AI工具平台构建的沉浸式法律 AI 应用

本项目聚焦当前法律行业中“案件增多、服务资源不均、普通用户庭前准备能力不足、法律训练场景沉浸感不强”等现实问题，围绕案件录入、庭审模拟、对方行为预判、胜诉率分析与复盘报告等核心环节，构建一套可交互、可解释、可演示的智慧法律系统。

## 2. 项目需求分析

当前法律服务与法律学习场景存在三个明显痛点。

第一，普通用户在遇到民间借贷、劳动争议、离婚纠纷、侵权责任等常见案件时，通常不知道案件的核心争点是什么，也无法判断自己手里的证据是否足够，导致庭前准备效率低、方向散。

第二，法律学习与模拟训练场景往往停留在静态文本层面，难以形成“案件推进-对方抗辩-证据补强-风险反馈”的动态体验，无法有效帮助学生和初级法律工作者建立真实的庭审思维。

第三，现有法律 AI 产品虽然在检索、问答、文书生成等方面已有进展，但在“沉浸式庭审互动”和“结构化庭前推演”方面仍存在较大提升空间。

因此，本项目希望解决的问题是：

- 帮助用户快速识别案件争点
- 帮助用户判断证据缺口
- 帮助用户低成本体验庭审推进过程
- 帮助用户获得更直观的胜诉率分析与复盘建议

## 3. 解决方案思路

本项目采用“前端交互层 + 元器工作流编排层 + 后端控制层 + 法律原子能力层”的整体架构。

其中，前端负责案件录入、互动模拟和分析展示；腾讯元器负责智能体工作流编排；得理开放平台API负责提供法规与案例检索原子能力；小理AI负责提供法律实务应用参考与表达辅助；后端负责状态管理、规则约束、报告生成与结果归档；腾讯开悟比赛平台承载赛题链路打通、资料支持与提交流程；CodeBuddy承担代码开发与工程协同；腾讯元宝用于 Prompt 优化与材料文本润色。

通过这种方式，项目不把自己做成一个泛法律聊天机器人，而是做成一个面向庭审模拟与法律辅助分析场景的系统解决方案。

## 4. 关键功能说明

### 4.1 案件信息可视化录入

用户可以录入案件类型、当事人关系、核心事实、已有证据和预期目标，系统将把自然语言案件描述转化为结构化案件信息，为后续模拟和分析提供基础。

### 4.2 庭审模拟交互

系统以沉浸式交互方式推进庭审流程，围绕举证、质证、法庭辩论等关键阶段生成场景文本和策略分支，使用户能够在可控结构中体验案件推进过程。

### 4.3 对方行为模拟

系统能够模拟对方当事人、代理人或证人的典型抗辩和行为变化，增强庭审训练的真实性与不确定性。

### 4.4 胜诉率分析与复盘报告

系统结合案件争点、证据情况、法规与类案检索结果，对案件进行分维度分析，并输出胜诉率判断、风险提示、补证建议和复盘报告。

## 5. 项目使用场景

本项目主要面向以下三类场景：

- 法律专业学生进行模拟庭审训练
- 普通用户进行庭前准备和法律问题初步梳理
- 法律服务辅助场景中的案件预分析与展示

在这些场景中，系统的价值不在于替代律师或法官，而在于帮助用户更低门槛地理解案件、梳理争点、准备证据并形成更清晰的策略意识。

## 6. AI工具使用说明（工作流与Prompt）

本项目严格依据赛题要求，使用以下官方工具完成设计、开发、联调与展示：

- `腾讯开悟比赛平台`
- `CodeBuddy`
- `腾讯元器`
- `腾讯元宝`
- `小理AI`
- `得理开放平台API`

在具体分工上：

- 腾讯元器作为智能体工作流编排主载体
- 得理开放平台API提供案例与法规检索能力
- 小理AI提供法律实务应用参考与辅助分析
- 腾讯元宝用于 Prompt 调优和文本表达优化
- CodeBuddy用于代码开发与工程协同
- 腾讯开悟比赛平台用于赛事链路打通与提交展示

本项目的核心工作流包括：

1. 案件信息输入
2. 案件类型与争点识别
3. 法规与案例检索
4. 当前庭审阶段内容生成
5. 对方行为模拟
6. 结果聚合与风险提示
7. 胜诉率分析与复盘输出

在 Prompt 设计上，本项目强调以下原则：

- 法律专业性优先
- 输出结构化优先
- 不确定时给出风险提示
- 不输出替代律师或法官的绝对性结论

## 7. 本项目的创新点总结

与常规法律问答系统相比，本项目的创新主要体现在：

- 将法律 AI 能力嵌入沉浸式庭审模拟场景
- 将法规检索、案例检索、对方行为模拟与胜诉率分析串成一条完整链路
- 强调可解释、可交互、可复盘，而非单次问答输出
- 结合企鹅 IP 与文游式体验，提升展示效果和传播辨识度

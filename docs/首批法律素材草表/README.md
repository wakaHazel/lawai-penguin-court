## 使用方式

1. 法学同学先通读，不要急着重写
2. 直接在原文上标出：
   - 哪些点成立
   - 哪些点表述不严谨
   - 哪些法条依据需要替换或补足
   - 哪些证据项需要增删
3. 优先把“是否真实、是否能用于比赛演示、是否符合法律逻辑”钉死

## 审校优先级

1. 争点是否抓对
2. 抗辩是否真实
3. 证据链是否合理
4. 法律依据是否需要修正到具体条文
5. 是否适合拿去做元器演示样例

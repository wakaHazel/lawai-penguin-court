# 得理检索工具配置清单（元器）

## 需要手工建的 3 个工具

1. `search_case`
2. `search_law`
3. `get_law_info`

## search_case

- Method：`POST`
- URL：`https://openapi.delilegal.com/api/qa/v3/search/queryListCase`
- Header：`appid`、`secret`

```json
{
  "pageNo": 1,
  "pageSize": 5,
  "sortField": "correlation",
  "sortOrder": "desc",
  "condition": {
    "keywordArr": ["{{keyword}}"]
  }
}
```

注意：`keywordArr` 必须是数组。

## search_law

- Method：`POST`
- URL：`https://openapi.delilegal.com/api/qa/v3/search/queryListLaw`
- Header：`appid`、`secret`

```json
{
  "pageNo": 1,
  "pageSize": 5,
  "sortField": "correlation",
  "sortOrder": "desc",
  "condition": {
    "keywords": ["{{keyword}}"],
    "fieldName": "semantic"
  }
}
```

## get_law_info

- Method：`GET`
- URL：`https://openapi.delilegal.com/api/qa/v3/search/lawInfo?lawId={{law_id}}&merge=true`
- Header：`appid`、`secret`

## 使用顺序

1. 先单测 3 个工具都可用
2. 再把它们接进工作流
3. 法规列表不够时，再补 `get_law_info`

# 元器最小回传样例需求（变量面板）

为了继续修 `工作流级变量 / 启动工作流输入变量` 的导入兼容问题，请额外准备一个最小元器导出样例。

## 最小样例要求

画布只放：

1. `开始节点`
2. `代码节点`
3. `结束节点`

## 手工配置

开始节点参数：

- `case_id = CASE-001`
- `branch_focus = 劳动关系是否成立`

工作流级变量：

- `querytips = 本工作流仅用于案例法规检索`
- `debug_flag = on`

## 代码节点

```python
def main(params: dict) -> dict:
    return {
        "case_id": params.get("case_id", ""),
        "branch_focus": params.get("branch_focus", ""),
        "querytips": params.get("querytips", ""),
        "debug_flag": params.get("debug_flag", "")
    }
```

## 结束节点输出

绑定：

- `case_id`
- `branch_focus`
- `querytips`
- `debug_flag`

## 导出前先看 UI

必须先确认元器界面里已经能看到这些变量，再导出。

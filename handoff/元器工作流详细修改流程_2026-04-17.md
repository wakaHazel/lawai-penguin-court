### 6. `W00.代码2` 改成“取第一个非空结果”，不要再直接拼接四路字符串

节点：

1. 工作流：`W00_企鹅法庭主控编排`
2. 节点名：`代码2`
3. 节点 ID：`f2e6c6e7-53c3-6103-78f2-d9d867369ef6`

当前代码有问题：

```python
def main(params):
    s1 = str(params.get("s1") or "")
    s2 = str(params.get("s2") or "")
    s3 = str(params.get("s3") or "")
    s4 = str(params.get("s4") or "")
    
    b1 = str(params.get("b1") or "")
    b2 = str(params.get("b2") or "")
    b3 = str(params.get("b3") or "")
    b4 = str(params.get("b4") or "")
    
    return {
        "final_out": s1 + s2 + s3 + s4,
        "branch_name": b1 + b2 + b3 + b4
    }
```

为什么不行：

1. 分支是互斥的，不应该拼接
2. 以后如果某次某个分支返回空、另一个分支也有残值，会很难排查

改成下面这版：

```python
def main(params):
    candidates = [
        (str(params.get("s1") or ""), str(params.get("b1") or "")),
        (str(params.get("s2") or ""), str(params.get("b2") or "")),
        (str(params.get("s3") or ""), str(params.get("b3") or "")),
        (str(params.get("s4") or ""), str(params.get("b4") or "")),
    ]

    for text, branch in candidates:
        if text:
            return {
                "final_out": text,
                "branch_name": branch or "unknown"
            }

    return {
        "final_out": "⚠️ 报错：工作流运行完成，但没有生成可回复内容。",
        "branch_name": "empty"
    }
```

### 8. `W02.代码1` 不能只按逗号切分，必须先尝试解析 JSON 数组

节点：

1. 工作流：`W02_法律支持检索`
2. 节点名：`代码1`
3. 节点 ID：`5dc47a01-3fb2-77bb-9623-2afb4aec3f75`

当前代码的问题：

1. 只把输入当逗号分隔字符串处理
2. 但从 `W00` 传过来的 `v_focus_issues` 和 `v_fact_keywords` 实际上经常是 JSON 数组字符串
3. 例如：

```text
["劳动关系是否成立","工资支付证据是否充分"]
```

当前代码会切成带引号和方括号的脏字符串，不是干净关键词。

改成下面这版：

```python
import json

def normalize_list(value, default_values):
    if value is None:
        return default_values

    if isinstance(value, list):
        result = [str(x).strip() for x in value if str(x).strip()]
        return result or default_values

    text = str(value).strip()
    if not text:
        return default_values

    try:
        parsed = json.loads(text)
        if isinstance(parsed, list):
            result = [str(x).strip() for x in parsed if str(x).strip()]
            return result or default_values
        if isinstance(parsed, str) and parsed.strip():
            return [parsed.strip()]
    except Exception:
        pass

    text = text.replace('，', ',')
    raw_list = text.split(',')
    cleaned = []
    for item in raw_list:
        item = item.strip().strip('\"').strip(\"'\").strip()
        item = item.strip('[]').strip()
        if item:
            cleaned.append(item)

    return cleaned or default_values


def main(params):
    issues_in = params.get('issues_in', '')
    keywords_in = params.get('keywords_in', '')

    issues_out = normalize_list(issues_in, ["民间借贷纠纷"])
    keywords_out = normalize_list(keywords_in, ["民间借贷"])

    return {
        "issues_out": issues_out,
        "keywords_out": keywords_out
    }
```
